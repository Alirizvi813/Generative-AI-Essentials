# Generative AI Text Generation Assignment

## Student
Ali Raza

## Project Overview
This project explains Generative AI and GPT concepts and implements a basic text generation model using Python and Keras.

## Dataset
A small public-domain style text dataset inspired by Project Gutenberg literature is included directly in the notebook. This avoids download errors in Google Colab.

## Main Features
- Environment setup and imports
- Tokenizer creation and model loading/building
- GPT architecture explanation
- Basic text generation
- Output length control
- Temperature adjustment
- Top-k and top-p sampling
- Prompt engineering examples
- Practical application demonstration
- Ethical considerations
- Report and visualizations

## How to Run in Google Colab
1. Upload `Generative_AI_Text_Generation_Ali_Raza_executed.ipynb` to Google Colab.
2. Run each cell from top to bottom.
3. If TensorFlow is not available, uncomment the install line in the first code cell.
4. Review generated text examples and training visualizations.
